# Servo

## Quick facts
- Mozilla によって開始され、現在は Linux Foundation がホストする Rust 製ブラウザエンジン。
- メモリ安全性と並列処理をコア・バリューに置く次世代アーキテクチャ。
- Chromium とは全く異なるコードベース。

## 主要構造とファイル
- **リポジトリ**: `repos/_browser/servo`
- **コンポーネント**: `components/`
    - `script/`: DOM, JavaScript 実行（SpiderMonkey 連携）
    - `layout/`: レイアウト計算
    - `style/`: CSS 解析（Stylo として Firefox にも提供）
    - `net/`: ネットワークスタック
- **ビルドツール**: `mach` (Python 製のビルドオーケストレーター)

## 全体アーキテクチャ

### 並列レンダリング
Servo の最大の特徴は、ブラウザの各工程（解析、レイアウト、描画）を独立したタスクとして並列実行することにある。
- 従来のエンジンがシングルスレッドのボトルネックに悩まされるのに対し、Rust の安全な並列性を活かしてマルチコア CPU を最大限活用する。

### WebRender
GPU を活用した超高速な描画エンジン。Web ページをゲームのレンダリングのように扱い、レイヤーの合成処理を GPU にオフロードする。

## 独自技術・アプローチ
- **Rust によるメモリ安全性**: セグメンテーションフォールトやデータ競合による脆弱性を言語レベルで排除。
- **コンポーネントの再利用**: Servo の一部（Stylo 等）は Firefox に統合されており、実戦投入も進んでいる。
